# AI SaaS Frontend

React + Vite frontend for the AI SaaS chat application.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The dev server will start on `http://localhost:5173` and proxy API calls to the backend on `http://localhost:8000`.

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
